# MADLM

This repository contains the MATLAB code and example data for the MADLM model.

MADLM is an advanced dorsiventral leaf radiative transfer model for simulating multi-angular and spectral reflection, considering the asymmetry of leaf internal and surface structures.

sMADLM is a simplified version of MADLM designed for efficient simulation and inversion using close-range hyperspectral images. It retains the representation of dorsiventral leaf optical properties while reducing model complexity for image-based applications.

## Model description

The models in this repository include:

1. **MADLM**: A dorsiventral leaf radiative transfer model for simulating multi-angular bidirectional reflectance of adaxial and abaxial leaf sides.
2. **sMADLM**: A simplified dorsiventral leaf radiative transfer model for close-range hyperspectral image applications and biochemical trait retrieval.
